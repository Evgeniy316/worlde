Лабораторная работа №42 — аналог игры Wordle

Описание
Приложение реализует игру Wordle: нужно угадать загаданное слово из 5 букв за 6 попыток.
После каждой попытки буквы в сетке подсвечиваются:
- зелёный: буква на своём месте
- жёлтый: буква есть в слове, но стоит не на своём месте
- серый: буквы нет в загаданном слове

Функциональность
- Стартовый экран с кнопкой «Начать игру»
- Игровой экран с полем 6×5, кнопкой «Проверить», сообщениями о победе/поражении и кнопкой «Новая игра»
- Ввод слова через виртуальную клавиатуру (русские буквы)
- Ограничение ввода: максимум 5 букв; после ввода 5 букв клавиатура блокируется до удаления/очистки/проверки
- Серые (отсутствующие) буквы подсвечиваются серым на виртуальной клавиатуре

Архитектура
- MVVM: логика игры и состояние находятся в GameViewModel (LiveData), UI наблюдает за состоянием
- Навигация построена через Navigation Component (nav_graph.xml)
- Используется Data Binding (buildFeatures.dataBinding = true)
